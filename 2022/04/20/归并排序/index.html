归并排序

Posted on 2022-04-20 Edited on 2022-07-02 In 算法
author:forxq

归并排序（英语：merge sort）是一种采用了 分治 思想的排序算法。

步骤 ：

step1: 将数列从中间划分为两部分

step2: 调用递归将左右两个子数列继续进行归并排序

step3:将两个有序的子序列合并成整个完整数列**（双指针）

时间复杂度 ：归并排序的最优时间复杂度、平均时间复杂度和最坏时间复杂度均为 O（nlogn）。

每递归一层， 每个数列会产生两层子数列

空间复杂度 O(n)

步骤1 取中间

int mid = (left+right)/2;
步骤2 左右两个子数列继续归并

merge_sort(a,left,mid);
merge_sort(a,mid+1,right);

步骤3 合并有序子数列

int i = left , k = 0 ,j = mid + 1;
while (i<=mid && j <= right){
    if (a[i]<=a[j])
        tmp[k++] = a[i++];
    else 
        tmp[k++] = a[j++];
}

while (i<=mid)    tmp[k++] = a[i++];
while (j<=right) 	tmp[k++] = a[j++];
